AOBD Orechunks

This Minecraft mod allows you to use fortune on smeltable ores so you can get more out of your fortune enchantment.

This mod does pretty much the exact same thing as Fortune ores and it's also heavily inspired by it. When you install this mod, you don't have to get rid of fortune ores, you can simply keep it if you want. If you don't want to keep using fortune ores I guess you can replace it with this mod.

This mod is heavily based on JAOPCA Singularities.
